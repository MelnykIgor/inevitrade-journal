# MelFemTrade — Trading Journal

React + Tailwind CSS, фіолетова тема, дані зберігаються в localStorage браузера.

## Запуск

```bash
npm install
npm run dev
```

Відкрити вказаний у консолі URL (зазвичай http://localhost:5173).

## Збірка для продакшну

```bash
npm run build
npm run preview
```

## Можливості
- Dashboard зі статистикою: Total Trades, Win Rate, Net P&L, Biggest Win, Biggest Loss, Win/Loss
- Таблиця угод (дата, пара, TF, позиція, результат, P&L, RR, сесія)
- Форма додавання нової угоди
- Видалення угод
- Збереження даних у localStorage
- Адаптивний дизайн (мобільні / планшети / десктоп)
